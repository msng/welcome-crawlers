Hides "Search Engine Visibility" section and turns off the checkbox to always welcome search engines' crawlers

== Description ==

Hides "Search Engine Visibility" section from Setting -> Reading
and always turns off the "Discourage search engines from indexing this site" checkbox
to welcome search engines' crawlers.

= How to Use =

Just install.

== Installation ==

1. Upload `welcome-crawlers` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
